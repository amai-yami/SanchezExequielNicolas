# Trabajo Anual - Desarrollo Web

## Descripción

Este repositorio reúne tres proyectos desarrollados como trabajo anual para la carrera cursada en el **Instituto Sedes Sapientiae**.

Los proyectos fueron realizados con el objetivo de aplicar los conocimientos adquiridos durante la cursada, implementando aplicaciones web con funcionalidades CRUD, manejo de bases de datos, validaciones y lógica de negocio.

## Tecnologías utilizadas

* PHP
* HTML5
* CSS3
* JavaScript
* MySQL
* Laragon (entorno de desarrollo)

## Contenido del repositorio

El repositorio contiene tres proyectos independientes desarrollados durante el año académico, cada uno enfocado en diferentes aspectos del desarrollo web utilizando PHP y MySQL.

## Objetivo

Aplicar conceptos de:

* Programación web.
* Arquitectura de aplicaciones.
* Gestión de bases de datos.
* Operaciones CRUD.
* Validaciones.
* Desarrollo de interfaces web.
